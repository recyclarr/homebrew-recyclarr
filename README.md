# Homebrew Tap for Recyclarr

Official Homebrew tap for [Recyclarr](https://github.com/recyclarr/recyclarr), a CLI tool to sync
TRaSH Guides to Sonarr/Radarr.

## Installation

```bash
brew tap recyclarr/recyclarr
brew install recyclarr
```

## Updating

```bash
brew update
brew upgrade recyclarr
```
